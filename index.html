CSS

Enter your CSS notes here

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS defines how a webpage looks
  • CSS Code added alongside HTML code is called inline css
  • CSS Code added to the head element of a webpage is called internal stylesheet
  • CSS Code added to a seperate sheet linked to the webpage is called external stylesheet
  • All 3 types are acceptable but external stylesheet is best practice because it is more flexable than the other two types of CSS/
